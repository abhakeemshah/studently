# Study Flashcards

A polished, interactive study hub with **97 Urdu→English vocabulary flashcards**, **70 GK MCQs**, **historical milestones**, and an **annual calendar**. Built as a single HTML file — no build tools needed.

## Features

- **📖 Vocabulary** — 97 words with Urdu text, English translation, memory hooks, and flip-to-reveal
- **🧠 GK MCQs** — 70 general knowledge questions with flip-to-reveal answers
- **📜 Historical Timeline** — Horizontal scroll through key world events
- **📅 Annual Calendar** — 365-day view with event markers
- **🌙 Dark/Light theme** — Persists via `localStorage`
- **📱 Responsive** — Works on mobile (320px) to desktop (1920px+)
- **⌨️ Keyboard nav** — Arrow keys + Space/Enter to flip
- **👆 Touch swipe** — Swipe left/right to navigate cards
- **📊 Progress bars** — See how far you've gone in each module

## How to Use

1. Open `index.html` in any browser
2. Tap the **nav tabs** to switch between modules
3. **Tap** a card to flip it and reveal the answer
4. Click **← Prev / Next →** to navigate
5. Toggle **🌙 / ☀️** for dark/light mode

## Design

- **Colors**: Indigo primary (`#6366f1`), amber accent (`#f59e0b`)
- **Typography**: Inter + Noto Nastaliq Urdu + JetBrains Mono
- **Animations**: `cubic-bezier(0.4, 0.0, 0.2, 1)` for smooth card flips
- **Glassmorphism**: `backdrop-filter: blur(16px)` on nav and theme button

Built with ❤️ — no frameworks, no build step, just HTML/CSS/JS.
